Arcus-3D-M2
=====
MachineKit configuration files for the Arcus-3D-M2 printer prototype.
CRAMPS compatible, Linear Delta, Velocity extrusion, 6 color active mixing, Machineface UI. 

clone with
```
git clone https://github.com/Arcus-3d/Arcus-3D-M2.git --recursive
```

run with
```
cd Arcus-3D-M2
mklauncher .
```
